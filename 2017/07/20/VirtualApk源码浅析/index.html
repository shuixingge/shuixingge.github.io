如果对Activity启动流程比较熟悉的时候，可以知道startAcitvity一定会走到Instrumentation的execStartActivity,由于选了hook了Instrumentation,所以会走到,VAInstrumentation.execStartActivity();

  • transformIntentToExplicitAsNeeded: 当activity为隐式启动时，其component为空,这个方法的作用就是根据intent设置的action,category等信息去插件中匹配中的acitivity，然后设置component，简单说就是将隐式启动转为显示启动。

private void hookSystemServices() {
try {
Singleton defaultSingleton = (Singleton) ReflectUtil.getField(ActivityManagerNative.class, null, “gDefault”);
IActivityManager activityManagerProxy = ActivityManagerProxy.newInstance(this, defaultSingleton.get());

        // Hook IActivityManager from ActivityManagerNative
        ReflectUtil.setField(defaultSingleton.getClass().getSuperclass(), defaultSingleton, "mInstance", activityManagerProxy);

        if (defaultSingleton.get() == activityManagerProxy) {
            this.mActivityManager = activityManagerProxy;
        }
    } catch (Exception e) {
        e.printStackTrace();
    }
}
